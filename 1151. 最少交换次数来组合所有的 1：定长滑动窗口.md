题目简述：

> 在本题中存在一个二进制数组（只包含 0 和 1）。 你可以交换数组的任意元素。
>
> 现在需要你将数组中所有 1 进行组合。返回组合所需要的最少交换次数。

题目链接：[1151. 最少交换次数来组合所有的 1](https://leetcode.cn/problems/minimum-swaps-to-group-all-1s-together/description/)

LintCode 题目链接：[3675 · 最少交换次数来组合所有的 1](https://www.lintcode.com/problem/3675/)

# 思路

考虑滑动窗口。

首先，数组内 0 和 1 的数量是恒定的，因此经组合后连续的 1 含有多少个 1 也是可以被确定的。不妨记数组中共有 $x$ 个 1，则考虑原数组的任何一个长为 $x$ 的子数组，如果子数组中含有 $t$ 个 1，则相应地就含有 $x-t$ 个 0，那么我们至少需要经过 $x-t$ 次交换，才能使得子数组变为 1 的组合，即使得子数组内均为 1。

因此，我们考虑长为 $x$ 的定长滑动窗口，使滑动窗口从原数组的起点遍历至终点，就可以得到每个长为 $x$ 的子数组作为 1 的组合所需的最少交换次数，最终能够得到全局最少交换次数。

# 代码

算法的时间复杂度为 $O(n)$，空间复杂度为 $O(1)$。

```java
public class Solution {
    /**
     * @param arr: the binary array
     * @return: the minimal number of  swaps
     */
    public int minSwaps(int[] arr) {
        int x = 0;
        for (int a : arr) x += a;

        // init
        int t = 0;
        for (int i = 0; i < x; i++) t += arr[i];
        int res = x - t;

        for (int i = 1; i <= arr.length - x; i++) {
            if (arr[i - 1] == 1) t--;
            if (arr[i + x - 1] == 1) t++;
            res = Math.min(res, x - t);
        }

        return res;
    }
}
```
