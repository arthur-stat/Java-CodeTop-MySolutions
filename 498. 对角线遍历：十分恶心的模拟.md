题目简述：

> 给你一个大小为 `m x n` 的矩阵 `mat` ，请以对角线遍历的顺序，用一个数组返回这个矩阵中的所有元素。

题目链接：[498. 对角线遍历](https://leetcode.cn/problems/diagonal-traverse/)

# 前言

# 代码

```java
class Solution {
    public int[] findDiagonalOrder(int[][] mat) {

        int[] res = new int[mat.length * mat[0].length];
        int x = 0;
        int y = 0;
        int n = 0;

        while (n < res.length) {

            // ↗
            if (((x + y) & 1) == 0) {
                res[n] = mat[y][x];

                if (x + y <= mat[0].length - 1) {
                    while (y != 0) {
                        x++;
                        y--;
                        n++;
                        res[n] = mat[y][x];
                    }
                } else {
                    while (x != mat[0].length - 1) {
                        x++;
                        y--;
                        n++;
                        res[n] = mat[y][x];
                    }    
                }

                if (x < mat[0].length - 1) {
                    x++;
                    n++;
                } else {
                    y++;
                    n++;
                }
            } else {  // else if (((x + y) & 1) == 1) {
                // ↙
                res[n] = mat[y][x];

                if (x + y <= mat.length - 1) {
                    while (x != 0) {
                        x--;
                        y++;
                        n++;
                        res[n] = mat[y][x];
                    }
                } else {
                    while (y != mat.length - 1) {
                        x--;
                        y++;
                        n++;
                        res[n] = mat[y][x];
                    }    
                }

                if (y < mat.length - 1) {
                    y++;
                    n++;
                } else {
                    x++;
                    n++;
                }
            }
        }

        return res;
    }
}
```
