题目简述：

> 给定一个整数数组 `prices`，其中 `prices[i]`表示第 `i` 天的股票价格 ；整数 `fee` 代表了交易股票的手续费用。
>
> 你可以无限次地完成交易，但是你每笔交易都需要付手续费。如果你已经购买了一个股票，在卖出它之前你就不能再继续购买股票了。
>
> 返回获得利润的最大值。
>
> **注意：**这里的一笔交易指买入持有并卖出股票的整个过程，每笔交易你只需要为支付一次手续费。
>
> **提示：**
>
> - `1 <= prices.length <= 5 * 10^4`
> - `1 <= prices[i] < 5 * 10^4`
> - `0 <= fee < 5 * 10^4`

题目链接：[714. 买卖股票的最佳时机含手续费](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-with-transaction-fee/)

# 绪言

***买卖股票这一系列问题，都是 DP 或贪心算法的思路。***

# 动态规划

## 初步思路

对于每段递增的股票走势，如果不存在手续费，我们只需要在递增起始点买入、在递增结束点卖出，就总能得到最大利润。但当存在手续费时，问题会稍微复杂些，相当于为我们的最优化目标函数增加了罚函数项。

对于买卖股票这一系列问题，可以优先考虑 DP。设 $dp[i][j]$ 表示在前 $i$ 天内进行 $j$ 次交易且最终未持有任何股票时的最大利润，则

$$
dp[i][j]=\max\Big\{dp[i-1][j],\ \max\big\{dp[k][j-1]+\mathrm{prices}[i]-\min_{k<u<i}\big\{\mathrm{prices}[u]\big\}-\mathrm{fee}:k<i\big\}\Big\}
$$

其中 $u$ 代表新一笔交易卖出日，这里交易指一只股票完整的买入与卖出动作。

这个状态转移方程的含义是：如果希望在前 $i$ 天内进行 $j$ 笔交易以获得最大利润，要么在第 $i$ 天前就已完成了 $j$ 笔交易、在第 $i$ 天不进行交易，要么在第 $i$ 天前完成了 $j-1$ 笔交易且在第 $i$ 天选择卖出所持有的股票以完成第 $j$ 笔交易。

## 改进思路

上式稍显复杂，更重要的是按照上式实现的算法时间复杂度太高，因为在上式的设计中，***卖出时必须要知道买入价才能在卖出的那一刻结算利润***。

改进思路是：买卖次数这一状态的设计比较糟糕，一个关键点在于我们实际上无需关心具体的交易次数、具体在哪一天做了什么交易。如果按上式实现算法，则必须枚举交易次数，而每一次枚举交易次数状态的代价都并不小。所以，最好是在买入时就立刻结算利润，这样就不再需要历史交易信息，于是便能够在定义状态时避免引入交易次数。

考虑定义 $dp[i][j]$ 表示在第 $i$ 天时是否持有股票分别能获得的最大利润，不妨以 $j=0$ 表示未持有股票、$j=1$ 表示持有一支股票，则

$$
dp[i][0]=\max\big\{dp[i-1][0],\ dp[i-1][1]+\mathrm{prices}[i]-\mathrm{fee}\big\}
$$

该式的含义是：在第 $i$ 天未持有一支股票情况下能获得的最大利润，要么等于在第 $i-1$ 天未持有一支股票时能获得的最大利润（即在第 $i$ 天不做买入动作），要么等于在 $i$ 天卖出一支过去已持有股票后的最大利润。

$$
dp[i][1]=\max\big\{dp[i-1][1],\ dp[i-1][0]-\mathrm{prices}[i]\big\}
$$

该式的含义是：在第 $i$ 天持有一支股票情况下能获得的最大利润，要么等于在第 $i-1$ 天持有一支股票时能获得的最大利润（即在第 $i$ 天不做买入动作），要么等于在第 $i-1$ 天未持有股票且在第 $i$ 天买入股票时能获得的最大利润。

> 由于买入与卖出存在手续费，因此当天卖出然后立刻买入是没有意义的，这里便不需要考虑这一情况，即可以认为每一天要么买入、要么卖出、要么无动作。

> 此处的另一个关键点是在买入时就立刻更新「利润」，对资金减去买入时的股价，而不要在卖出时统一结算（卖出时只额外计算手续费即可）。如果不这样做，为了在卖出时计算利润就必须知道上一次买入的时间，于是便陷入了困境，无法设计出高效的状态转移。

## 代码实现

算法的时间复杂度为 $O(n)$，空间复杂度为 $O(1)$，其中运用了滚动数组的小技巧使空间开销来到常数数量级。

```java
class Solution {
    public int maxProfit(int[] prices, int fee) {
        int dp_i_0 = 0;
        int dp_i_1 = -prices[0];

        for (int i = 1; i < prices.length; i++) {
            int tmp_dp_0 = Math.max(dp_i_0, dp_i_1 + prices[i] - fee);
            dp_i_1 = Math.max(dp_i_1, dp_i_0 - prices[i]);
            dp_i_0 = tmp_dp_0;
        }

        return dp_i_0;
    }
}
```
