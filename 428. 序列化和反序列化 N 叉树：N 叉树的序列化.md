题目简述：

> 序列化是将一个数据结构或对象转换成比特流的过程，以便将其存储在文件或内存缓冲区中，或通过网络连接链路传输，以便稍后在同一或另一计算机环境中重建。
>
> 设计一个算法来序列化和反序列化一个N叉树。一棵N叉树是一棵有根树，其中每个节点的子节点不超过N个。序列化/反序列化算法的实现方式没有限制。您只需要确保N叉树可以序列化为字符串，并且该字符串可以反序列化为原始树结构。

# 代码

感觉没什么好讲的，标准的层序遍历可以轻松解决。只需要记得在序列化的过程中记录一个节点的子节点数量，不然无法唯一确定反序列化。

在反序列化的时候，也按层序遍历的逻辑进行。使用一个队列记录 `Pair` 序列，确保 `Pair` 按 FIFO 的顺序输出，每个 `Pair` 都包含一个父节点和该节点的剩余子节点数量。这样，我们顺序遍历此前经层序遍历得到的字符串表示，每当反序列化出一个节点后都通过 `Pair` 队列找到该节点的父节点，同时更新父节点的剩余子节点数量。

算法的时间复杂度为 $O(n)$，空间复杂度为 $O(n)$。

```java
/**
 * Definition for Directed graph.
 * class DirectedGraphNode {
 *     int label;
 *     ArrayList<DirectedGraphNode> neighbors;
 *     DirectedGraphNode(int x) { label = x; neighbors = new ArrayList<DirectedGraphNode>(); }
 * };
 */

public class Solution {
    public String serialize(DirectedGraphNode root) {  
        if (root == null) return "";
        Deque<DirectedGraphNode> q = new ArrayDeque<>();
        q.offer(root);
        StringBuilder sb = new StringBuilder();

        while (!q.isEmpty()) {
            DirectedGraphNode node = q.poll();
            sb.append(node.label + ";" + node.neighbors.size() + "|");
            q.addAll(node.neighbors);
        }

        return sb.toString();
    }

    public DirectedGraphNode deserialize(String data) {
        if (data.isBlank()) return null;
        String[] nodes = data.split("\\|");
        int rootLabel = Integer.valueOf(nodes[0].split(";")[0]);
        int rootNei = Integer.valueOf(nodes[0].split(";")[1]);
        DirectedGraphNode root = new DirectedGraphNode(rootLabel);
        Deque<Pair> count = new ArrayDeque<>();
        count.offer(new Pair(root, rootNei));

        for (int i = 1; i < nodes.length; i++) {
            int nodeLabel = Integer.valueOf(nodes[i].split(";")[0]);
            int nodeNei = Integer.valueOf(nodes[i].split(";")[1]);
            DirectedGraphNode node = new DirectedGraphNode(nodeLabel);

            while (count.peek().child == 0) count.poll();
            count.peek().parent.neighbors.add(node);
            count.peek().child--;

            count.offer(new Pair(node, nodeNei));
        }

        return root;
    }

    private static class Pair {

        DirectedGraphNode parent;
        int child;

        Pair(DirectedGraphNode parent, int child) {
            this.parent = parent;
            this.child = child;
        }
    }
}
```
