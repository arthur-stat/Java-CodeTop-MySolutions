题目简述：

> **丑数** 就是只包含质因数 `2`、`3` 和 `5` 的 *正* 整数。
>
> 给你一个整数 `n` ，请你判断 `n` 是否为 **丑数** 。如果是，返回 `true` ；否则，返回 `false` 。

题目链接：[263. 丑数](https://leetcode.cn/problems/ugly-number/)

# 思路

用除法除掉所有包含的 2、3、5 质因子，直到不可再除，然后判断剩下的数是不是 1 就可以了。

# 代码

```java
class Solution {
    public boolean isUgly(int n) {
        if (n <= 0) return false;

        while (n > 1) {
            if ((n & 1) == 0) n = n >>> 1;
            else break;
        }

        while (n > 1) {
            if (n % 3 == 0) n /= 3;
            else break;
        }

        while (n > 1) {
            if (n % 5 == 0) n /= 5;
            else break;
        }

        return n == 1;
    }
}
```
