题目简述：

> 给定一个整数数组`prices`，其中第 `prices[i]` 表示第 *`i`* 天的股票价格 。
>
> 设计一个算法计算出最大利润。在满足以下约束条件下，你可以尽可能地完成更多的交易（多次买卖一支股票）:
>
> - 卖出股票后，你无法在第二天买入股票 (即冷冻期为 1 天)。
>
> **注意：**你不能同时参与多笔交易（你必须在再次购买前出售掉之前的股票）。

题目链接：[309. 买卖股票的最佳时机含冷冻期](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-with-cooldown/)

# 思路

<u>**股票系列问题，都考虑动态规划或贪心算法。**</u>其中动态规划通常都能解决这一系列问题，但贪心算法则只针对其中的部分特定问题。

做如下状态定义：

- $dp[i][0]$：第 $i$ 日未持有股票且可以买入股票、即未处于冷却期时能获得的最大利润；
- $dp[i][1]$：第 $i$ 日售出股票能获得的最大利润；
- $dp[i][2]$：第 $i$ 日处于冷却期时能获得的最大利润；
- $dp[i][3]$：第 $i$ 日持有股票能获得的最大利润，这包含了继续持有历史购入的股票以及新购入股票。

> 在本问题的众多解答中，针对状态值 $dp[i][j]$，更常见的定义是对 $j$ 只定义三种情况。这里我定义了四种情况，相当于将 “未持有股票” 这一状态拆分为了 “$dp[i][0]$ 整日至始至终未持有股票” 与 “$dp[i][1]$ 当日售出股票” 两种颗粒度更小的状态，这样也是可行的。只要推导出的状态转移方程无误，便同样能设计出正确的算法。

状态转移方程为：

- $dp[i][0]=\max\big\{dp[i-1][0],\ dp[i-1][2]\big\}$
- $dp[i][1]=dp[i-1][3]+\mathrm{prices}[i]$
- $dp[i][2]=dp[i-1][1]$
- $dp[i][3]=\max\big\{dp[i-1][3],\ dp[i-1][0]-\mathrm{prices}[i],\ dp[i-1][2]-\mathrm{prices}[i]\big\}$

> 在最初推导出的状态转移方程中，我遗漏了 $dp[i-1][2]\to dp[i][3]$ 这一状态转移的可能，导致我实现的算法不能按预期工作。$dp[i-1][2]\to dp[i][3]$ 的含义是冷却期一过就不加等待地立刻购入股票，这当然是被允许的。
>
> 可以参考我在代码注释中给出的状态机模型，对应 `2 -> (0, 3)` 的转移链。

# 代码

算法的时间复杂度为 $O(n)$，空间复杂度为 $O(1)$，其中使用了滚动数组的方法，没有存储全部历史状态值。

```java
class Solution {
    public int maxProfit(int[] prices) {
        /*
          dp[i][0] 第 i 日未持有股票且能够买入股票能获得的最大利润
          dp[i][1] 第 i 日售出股票（因此未持有股票）能获得的最大利润
          dp[i][2] 第 i 日未持有股票但不能买入股票（因为处于冷却期）能获得的最大利润
          dp[i][3] 第 i 日持有股票（包含当日买入与继续持有两种情况）能获得的最大利润
          ---------------------------------------------------------------------
          状态机：0 -> ... -> 0 -> 3 -> ... -> 3 -> 1 -> 2 -> (0, 3)
        */
        int dp_i_0 = 0;
        int dp_i_1 = Integer.MIN_VALUE;
        int dp_i_2 = Integer.MIN_VALUE;
        int dp_i_3 = -prices[0];

        int n = prices.length;
        for (int i = 1; i < n; i++) {
            int t_dp_i_0 = Math.max(dp_i_0, dp_i_2);
            int t_dp_i_1 = dp_i_3 + prices[i];
            int t_dp_i_2 = (dp_i_1 == Integer.MIN_VALUE) ? Integer.MIN_VALUE : dp_i_1;
            int t_dp_i_3 = (dp_i_2 == Integer.MIN_VALUE) ?
                    Math.max(dp_i_3, dp_i_0 - prices[i]) :
                    Math.max(dp_i_3, Math.max(dp_i_0 - prices[i], dp_i_2 - prices[i]));

            dp_i_0 = t_dp_i_0;
            dp_i_1 = t_dp_i_1;
            dp_i_2 = t_dp_i_2;
            dp_i_3 = t_dp_i_3;
        }

        return Math.max(dp_i_0, Math.max(dp_i_1, dp_i_2));
    }
}
```
