题目简述：

> 某班级 n 位同学的学号为 0 ~ n-1。点名结果记录于升序数组 `records`。假定仅有一位同学缺席，请返回他的学号。

题目链接：[LCR 173. 点名](https://leetcode.cn/problems/que-shi-de-shu-zi-lcof/)

# 代码

算法的时间复杂度为 $O(\log n)$，空间复杂度为 $O(1)$。

```java
class Solution {
    public int takeAttendance(int[] records) {
        int l = 0;
        int r = records.length - 1;

        while (l <= r) {
            int mid = l + (r - l) / 2;

            if (records[mid] == mid) {
                l = mid + 1;
            } else {
                r = mid - 1;
            }
        }
        return l;
    }
}
```
