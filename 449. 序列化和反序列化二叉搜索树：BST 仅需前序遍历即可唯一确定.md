# 思路

BST 有一个非常好的性质，就是只需要前序遍历就可以唯一确定其结构。

对于一般的二叉树，需要至少两种遍历顺序才能唯一确定其结构，比如前序遍历与中序遍历；对于一般的 N 叉树，需要记录每个节点的子节点数量再结合某种遍历顺序才能唯一确定其结构，比如层序遍历的过程中同时记录节点的值与子节点数目i。

但 BST 不需要这么麻烦，BST 只需要前序遍历的顺序就能唯一确定结构了，因为 BST 本身的性质补充了足够多的信息。

---

在反序列化时，利用 BST 的任何子结构均满足「左＜中＜右」的大小顺序，结合先序遍历的「中 → 左 → 右」的顺序，我们可以知道：对于当前节点，其右侧一段连续的小于他的节点均属于其左子树，下一段连续的大于他的节点均属于其右子树。

利用这一性质，我们就可以容易地设计子树递归反序列化了。

# 代码

`serialize` 的时间复杂度为 $O(n)$，额外空间复杂度为 $O(h)$，其中 $h$ 为树高；`deserialize` 使用二分查找快速寻找左右子树分界点，最坏时间复杂度为 $O(n\log n)$（对于平衡 BST，时间复杂度为 $O(n)$），空间复杂度为 $O(h)$，其中空间开销主要来自于递归调用栈。

如果不结合二分查找，则 `deserialize` 的最坏时间复杂度为 $O(n^2)$。

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */

public class Codec {

    // Encodes a tree to a single string.
    public String serialize(TreeNode root) {
        if (root == null) return "";
        Deque<TreeNode> stack = new ArrayDeque<>();
        StringBuilder sb = new StringBuilder();

        while (root != null || !stack.isEmpty()) {
            while (root != null) {
                sb.append(root.val).append('|');
                if (root.right != null) stack.push(root.right);
                root = root.left;
            }

            if (!stack.isEmpty()) root = stack.pop();
        }

        sb.setLength(sb.length() - 1);
        return sb.toString();
    }

    // Decodes your encoded data to tree.
    public TreeNode deserialize(String data) {
        if (data.isBlank()) return null;
        int[] vals = Arrays.stream(data.split("\\|")).mapToInt(Integer::parseInt).toArray();
        return build(vals, 0, vals.length - 1);
    }

    private TreeNode build(int[] vals, int start, int end) {
        if (start > end) return null;
        TreeNode node = new TreeNode(vals[start]);
        if (start == end) return node;

        int i = start + 1;
        int j = end;
        while (i <= j) {
            int mid = i + (j - i) / 2;
            if (vals[mid] < vals[start]) i = mid + 1;
            else j = mid - 1;
        }
        node.left = build(vals, start + 1, i - 1);
        node.right = build(vals, i, end);

        return node;
    }
}
```

# 利用 BST 值域的性质优化时间复杂度

实际上，`deserialize` 可以被改进至在任何情况下时间复杂度均为严格 $O(n)$。关键在于利用 BST 节点值域的性质，`build` 不再寻找先序遍历中左右子树的分界点，而使用值的上下界估计夹逼节点的值。本方案是理论最优解，十分巧妙。

例如：

```java
public class Codec {

    private int pos;

    // Encodes a tree to a single string.
    // serialize...

    // Decodes your encoded data to tree.
    public TreeNode deserialize(String data) {
        if (data == null || data.isEmpty()) return null;
        int[] vals = Arrays.stream(data.split("\\|")).mapToInt(Integer::parseInt).toArray();
        pos = 0;
        return build(vals, Integer.MIN_VALUE, Integer.MAX_VALUE);
    }

    private TreeNode build(int[] vals, int lower, int upper) {
        if (pos == vals.length) return null;
        int val = vals[pos];
        if (val < lower || val > upper) return null;
        pos++;

        TreeNode node = new TreeNode(val);
        node.left = build(vals, lower, val);
        node.right = build(vals, val, upper);
        return node;
    }
}
```

`pos` 指针只会访问所有节点各一次，每次 `pos` 移动后，在子递归调用中通过设置输入参数以更新的上下界估计。
