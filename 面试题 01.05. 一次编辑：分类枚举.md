题目简述：

> 字符串有三种编辑操作:插入一个英文字符、删除一个英文字符或者替换一个英文字符。 给定两个字符串，编写一个函数判定它们是否只需要一次(或者零次)编辑。

题目链接：[面试题 01.05. 一次编辑](https://leetcode.cn/problems/one-away-lcci/)

# 思路

虽然可以用动态规划算法考虑编辑距离（[72. 编辑距离](https://leetcode.cn/problems/edit-distance/)）求解，但由于本问题并不需要我们给出最少编辑距离，而只需要给出能否在一次编辑内得到结果，那么直接枚举可能性就可以了（零次编辑、一次替换与一次增删，其中增与删操作是对称的），无需从动态规划的角度考虑。

# 代码

算法的时间复杂度为 $O\big(\max(m,n)\big)$，空间复杂度为 $O(1)$。

```java
class Solution {
    public boolean oneEditAway(String first, String second) {
        // 零次编辑
        if (first.equals(second)) return true;

        // 替换一个字符
        if (first.length() == second.length()) {
            int diff = 0;
            for (int i = 0; i < first.length(); i++) {
                if (first.charAt(i) != second.charAt(i)) diff++;
                if (diff > 1) return false;
            }
            return true;
        }

        // 增删一个字符
        if (Math.abs(first.length() - second.length()) > 1) return false;
        String longer = (first.length() >= second.length()) ? first : second;
        String shorter = (first.length() < second.length()) ? first : second;
        if (longer.substring(1, longer.length()).equals(shorter)) return true;
        int i = 0;
        int j = 0;
        boolean found = false;
        while (i < longer.length() && j < shorter.length()) {
            if (longer.charAt(i) != shorter.charAt(j)) {
                if (!found) {
                    found = true;
                    i++;
                    continue;
                } else {
                    return false;
                }
            }
            i++;
            j++;
        }
        return true;
    }
}
```
