题目简述：

> 设想有个机器人坐在一个网格的左上角，网格 r 行 c 列。机器人只能向下或向右移动，但不能走到一些被禁止的网格（有障碍物）。设计一种算法，寻找机器人从左上角移动到右下角的路径。
>
> 网格中的障碍物和空位置分别用 `1` 和 `0` 来表示。
>
> 返回一条可行的路径，路径由经过的网格的行号和列号组成。左上角为 0 行 0 列。如果没有可行的路径，返回空数组。

题目链接：[面试题 08.02. 迷路的机器人](https://leetcode.cn/problems/robot-in-a-grid-lcci/)

# 思路

非常经典的一类动态规划问题。

机器人只能向下与向右移动时，路径构成 DAG，因此可以应用动态规划算法。如果机器人能向上下左右四个方向移动，就破坏了 DAG，动态规划便不能解决这类问题了。

姊妹问题：

  - [62. 不同路径]([62. 不同路径](https://leetcode.cn/problems/unique-paths/))：无障碍，求方案数
  - [63. 不同路径 II](https://leetcode.cn/problems/unique-paths-ii/)：有障碍，求方案数
  - 本问题：有障碍，要求构造出一条可行路径

要给出路径，只需要保留完整的 DP 表，然后在求解动态规划后循着 DP 表倒着找到逆状态转移即可。

# 代码

算法的时间复杂度为 $O(mn)$，空间复杂度为 $O(mn)$。如果允许直接修改输入数组，那么可以做到 $O(1)$ 的额外空间复杂度。

```java
class Solution {
    public List<List<Integer>> pathWithObstacles(int[][] obstacleGrid) {
        int m = obstacleGrid.length;
        int n = obstacleGrid[0].length;
        boolean[][] dp = new boolean[m][n];
        List<List<Integer>> paths = new ArrayList<>();

        // init
        if (obstacleGrid[0][0] == 1) return paths;
        dp[0][0] = true;
        for (int col = 1; col < n; col++) {
            if (obstacleGrid[0][col] == 0) dp[0][col] = true;
            else break;
        }
        for (int row = 1; row < m; row++) {
            if (obstacleGrid[row][0] == 0) dp[row][0] = true;
            else break;
        }

        for (int row = 1; row < m; row++) {
            for (int col = 1; col < n; col++) {
                dp[row][col] = (obstacleGrid[row][col] == 0) && (dp[row - 1][col] || dp[row][col - 1]);
            }
        }

        if (!dp[m - 1][n - 1]) return paths;

        paths.add(List.of(m - 1, n - 1));
        int i = m - 1, j = n - 1;
        while (i != 0 || j != 0) {
            if (i > 0 && dp[i - 1][j]) {
                i--;
            } else {
                j--;
            }
            paths.addFirst(List.of(i, j));
        }
        return paths;
    }
}
```
