题目简述：

> 给定一个 24 小时制（小时:分钟 **"HH:MM"**）的时间列表，找出列表中任意两个时间的最小时间差并以分钟数表示。

题目链接：[539. 最小时间差](https://leetcode.cn/problems/minimum-time-difference/)

# 直观方案：排序（比较排序）

## 思路

这是最直观的方案，我们直接将时间列表数组按其字面量排序，然后两两计算排序后数组内相邻的元素的距离，则该距离的最小值就是全局最小距离——可以用反证法证明全局最小距离必然也是某个排序后相邻元素的距离。

这里有几个值得注意的点：

- 对于 "HH:MM" 时间表示而言构成一个回环，事实上这是一个 1440 阶循环群（$24\times 60=1440$），因此假设我们的排序是按字面量升序进行的，则计算距离时应取绝对差和绝对差在群中的补（$1440-x$）的最小值，即两个元素在群间的最小正位移绝对值。
- 排序时，可以直接按字符串字面量排序。不难发现，字符串字典序就是 "HH:MM" 时间表示的字面量序。
- 记得要特殊处理排序后首个元素和最后一个元素，必须要考虑上他们间的距离，否则可能漏掉真正的解。

> 我们可以想象一个圆形的几何环，若干个 "HH:MM" 的时间表示对应的点离散地分布在环上。我们按其字面量进行排序，于是就等于得到了环上各点的某种拓扑序——无论是逆时针还是顺时针，总之我们能够朝某个方向完整地遍历所有点。这样就不难想到全局最小距离必然就是遍历过程某两个相邻的点的距离了。

## 代码

在考虑比较排序特别是快速排序、归并排序（及其变种）的情况下，算法的时间复杂度为 $O(n\log n)$、空间复杂度为 $O(\log n)$——这些复杂度都主要来自于排序算法。

```java
class Solution {
    public int findMinDifference(List<String> timePoints) {
        timePoints.sort(String::compareTo);
        int res = calc(timePoints.getFirst(), timePoints.getLast());
        for (int i = 0; i < timePoints.size() - 1; i++) {
            res = Math.min(res, calc(timePoints.get(i), timePoints.get(i + 1)));
        }
        return res;
    }

    public int calc(String t1, String t2) {
        int val = h2m(t2) - h2m(t1);
        return Math.min(val, 1440 - val);
    }

    public int h2m(String t) {
        return Integer.parseInt(t.substring(0, 2)) * 60 + Integer.parseInt(t.substring(3, 5));
    }
}
```

# 最佳方案：桶

## 思路

类似桶排序中对「桶」的处理，由于一天 24 时内的 "HH:MM" 表示中至多有 1440 种不同的字面量，因此可以使用一个大小为 1440 的数组作为桶，接着将所有的时间表示放入桶内，最后顺序遍历桶数组也能得到各时间表示的某种拓扑序。与上一种方法没有本质不同，但由于我们避免了直接进行排序，因此可以做到 $O(n)$ 的稳定理论时间复杂度。

<u>这里可以考虑一个剪枝：根据抽屉原理，如果时间列表长度大于 1440，则答案必然为 0。</u>

> 剪枝在上一步总当然也可以做，但我认为在桶的思路下会更加直白——虽然这个逻辑本身就非常简单。

## 代码

算法的时间复杂度为 $O(n)$，空间复杂度为 $O(n)$。

```java
class Solution {
    public int findMinDifference(List<String> timePoints) {
        if (timePoints.size() > 1440) return 0;  // 剪枝 1
        boolean[] exists = new boolean[1440];
        int minIndex = Integer.MAX_VALUE;
        int maxIndex = Integer.MIN_VALUE;
        for (String timePoint : timePoints) {
            int index = h2m(timePoint);
            if (exists[index]) return 0;  // 剪枝 2
            minIndex = Math.min(minIndex, index);
            maxIndex = Math.max(maxIndex, index);
            exists[index] = true;
        }

        int res = Math.min(maxIndex - minIndex, 1440 - (maxIndex - minIndex));
        int interval = 1;
        for (int i = minIndex + 1; i <= maxIndex; i++) {
            if (exists[i]) {
                res = Math.min(res, interval);
                interval = 1;
            } else {
                interval++;
            }
        }
        return res;
    }

    public int h2m(String t) {
        return Integer.parseInt(t.substring(0, 2)) * 60 + Integer.parseInt(t.substring(3, 5));
    }
}
```
