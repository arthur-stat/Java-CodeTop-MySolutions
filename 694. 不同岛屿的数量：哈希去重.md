题目简述：

> 给定一个由0和1组成的非空的二维网格，一个**岛屿**是指四个方向（包括横向和纵向）都相连的一组1（1表示陆地）。你可以假设网格的四个边缘都被水包围。
>
> 找出所有**不同**的岛屿的个数。如果一个岛屿与另一个岛屿形状相同（不考虑旋转和翻折），我们认为这两个岛屿是相同的。
>
> 注意：
>
> ```
> 11
> 1
> ```
>
> 和
>
> ```
>  1
> 11
> ```
>
> 是不同的岛屿，因为我们不考虑旋转和翻折。

题目链接：[694. 不同岛屿的数量](https://leetcode.cn/problems/number-of-distinct-islands)

LintCode 题目链接：[860 · 不同岛屿的个数](https://www.lintcode.com/problem/860)

# 思路

在 [200. 岛屿数量](https://leetcode.cn/problems/number-of-islands/) 的基础上，额外添加去重检查的逻辑即可。

至于如何去重的实现，我想到了两个方案：

- 方案一，为上下左右的探索赋予不同的互质权重系数，用 `int` 表示探索的哈希值，最后通过哈希值去重；
- 方案二，用字符串记录完整的探索路径，对字符串哈希去重。

两个方案应该都是可行的。下面的代码实现的是第二个方案。

# 代码

算法的时间复杂度为 $O(mn)$，空间复杂度为 $O(mn)$。

```java
public class Solution {
    /**
     * @param grid: a list of lists of integers
     * @return: return an integer, denote the number of distinct islands
     */
    public int numberofDistinctIslands(int[][] grid) {
        int count = 0;
        Set<String> set = new HashSet<>();
        for (int i = 0; i < grid.length; i++) {
            for (int j = 0; j < grid[0].length; j++) {
                if (grid[i][j] == 1) {
                    StringBuilder path = new StringBuilder();
                    dfs(grid, path, i, j, i, j);
                    if (set.add(path.toString())) count++;
                }
            }
        }
        return count;
    }

    private void dfs(int[][] grid, StringBuilder path, int x0, int y0, int x, int y) {
        if (x < 0 || y < 0 || x >= grid.length || y >= grid[0].length || grid[x][y] == 0) return;

        path.append("" + (x - x0) + (y - y0));
        grid[x][y] = 0;

        dfs(grid, path, x0, y0, x, y - 1);
        dfs(grid, path, x0, y0, x, y + 1);
        dfs(grid, path, x0, y0, x - 1, y);
        dfs(grid, path, x0, y0, x + 1, y);
    }
}
```
