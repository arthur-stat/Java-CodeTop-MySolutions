题目简述：

> 给定二叉树的根节点 `root` ，返回所有左叶子之和。

题目链接：[404. 左叶子之和](https://leetcode.cn/problems/sum-of-left-leaves/)

# 思路

DFS 或 BFS 时对叶子左子节点计数即可，即同时完成：

1. 判别是否为叶子节点；
2. 判别是否为左子节点。

# 代码

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */

class Solution {
    public int sumOfLeftLeaves(TreeNode root) {
        return dfs(root, 0);
    }

    /* sgn = -1 表示该节点为左子节点，sgn = 1 表示该节点为右子节点 */
    private int dfs(TreeNode root, int sgn) {
        if (root == null) return 0;

        int res = 0;
        if (root.left == null && root.right == null && sgn == -1) res += root.val;
        return dfs(root.left, -1) + dfs(root.right, 1) + res;
    }
}
```
