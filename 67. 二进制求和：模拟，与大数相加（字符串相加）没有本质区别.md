题目简述：

> 给你两个二进制字符串 `a` 和 `b` ，以二进制字符串的形式返回它们的和。

题目链接：[67. 二进制求和](https://leetcode.cn/problems/add-binary/)

# 代码

双指针直接模拟即可。

```java
class Solution {
    public String addBinary(String a, String b) {
        int i = a.length() - 1;
        int j = b.length() - 1;
        boolean carry = false;
        int len = Math.max(a.length(), b.length());
        char[] res = new char[len];
        int k = len - 1;

        while (i >= 0 && j >= 0) {
            int c = a.charAt(i) + b.charAt(j)- 2 * '0';
            if (carry) c++;

            char cc = switch (c) {
                case 0 -> {
                    carry = false;
                    yield '0';
                }
                case 1 -> {
                    carry = false;
                    yield '1';
                }
                case 2 -> {
                    carry = true;
                    yield '0';
                }
                case 3 -> {
                    carry = true;
                    yield '1';
                }
                default -> throw new IllegalArgumentException();
            };

            res[k--] = cc;
            i--;
            j--;
        }

        String c;
        int m;
        if (i >= 0) {
            c = a;
            m = i;
        } else {
            c = b;
            m = j;
        }

        while (m >= 0) {
            int cc = c.charAt(m) - '0';
            if (carry) cc++;

            char ccc = switch (cc) {
                case 0 -> {
                    carry = false;
                    yield '0';
                }
                case 1 -> {
                    carry = false;
                    yield '1';
                }
                case 2 -> {
                    carry = true;
                    yield '0';
                }
                default -> throw new IllegalArgumentException();
            };

            res[k--] = ccc;
            m--;
        }

        return carry ? "1" + new String(res) : new String(res);
    }
}
```
