题目简述：

> 递归乘法。 写一个递归函数，不使用 * 运算符， 实现两个正整数的相乘。可以使用加号、减号、位移，但要吝啬一些。

题目链接：[面试题 08.05. 递归乘法](https://leetcode.cn/problems/recursive-mulitply-lcci/)

# 代码

嗯……没什么好讲的，按二进制的竖式乘法来就好了。
$$
\begin{array}{r@{\qquad}r@{\ }rrrrrr}
(A) &&&& 1&0&1&1\\
(B) &\times&&&& 1&0&1\\
\hline
&&&& 1&0&1&1\\
&&& 0&0&0&0&\\
&& 1&0&1&1&&\\
\hline
&& 1&1&0&1&1&1
\end{array}
$$
这应该很明了吧，从竖式乘法上看二进制乘法就只包含取原值、取零值、加法和位移操作。

每个递归调用只计算中间式的某一行，让子递归计算剩余的行，最后在父递归中通过位移操作把子递归返回的中间结果以及当前递归体中算出的一行相加。

```java
class Solution {
    public int multiply(int A, int B) {
        if (B == 0) return 0;
        int res = ((B & 1) == 1) ? A : 0;
        return res + (multiply(A, B >> 1) << 1);
    }
}
```
