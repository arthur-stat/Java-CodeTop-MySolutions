# 对撞指针（不保序）

## 思路

参考快速排序算法的三向切分的核心——荷兰国旗问题，我们可以使用首尾指针（也称为对撞指针）解决这个问题。

该算法不保持其他元素的相对顺序，因为其主要动作是「交换」。如果要保序，应考虑主要动作为「覆盖」的快慢指针。

## 代码

算法的时间复杂度为 $O(n)$，系原地算法，仅 $O(1)$ 的额外空间开销。不保序。

```java
class Solution {
    public int removeElement(int[] nums, int val) {
        int i = nums.length - 1;
        int j = 0;

        while (j <= i) {
            if (nums[j] == val) {
                swap(nums, i, j);
                i--;
            } else {
                j++;
            }
        }

        return j;
    }

    public void swap(int[] nums, int i, int j) {
        int tmp = nums[i];
        nums[i] = nums[j];
        nums[j] = tmp;
    }
}
```

# 快慢指针（保序）

## 思路

核心思路依然是双指针，不过为了保序，原本在对撞指针中第一个指针用于指向当前访问元素、第二个指针用于指向下一个被移除元素需要被交换去的位置，那么在这里需要相应地修改为让第一个指针指向下一个有效元素的位置、第二个指针指向当前访问元素。

由于第一个指针表示下一个有效元素的位置，因此能够保证有效元素间的相对顺序不变。

## 代码

算法的时间复杂度为 $O(n)$，系原地算法，仅 $O(1)$ 的额外空间开销。保序。

```java
class Solution {
    public int removeElement(int[] nums, int val) {
        int i = 0;

        for (int j = 0; j < nums.length; j++) {
            if (nums[j] != val) nums[i++] = nums[j];
        }

        return i;
    }
}
```
