题目简述：

> 36进制数由 0~9 ， a~z 共 36 个字符组成，请你按照加法规则计算两个 36 进制数的和。
>
> 例如 1b+2b = 3m
>
> 数据范围：两个数的长度都满足 $1\leqslant n\leqslant 10^6$ ， 两个数中包含的字符仅包括 $0−9$ 和 $a−z$

题目链接：[36进制加法](https://www.nowcoder.com/practice/c5db069fd9d64e6e9cf5fd68860abcdd)

# 思路

模拟，和字符串加法（大数相加）一样。

# 代码

```java
import java.util.*;


public class Solution {
    /**
     * 代码中的类名、方法名、参数名已经指定，请勿修改，直接返回方法规定的值即可
     *
     * 
     * @param A string字符串 
     * @param B string字符串 
     * @return string字符串
     */
    public String thirtysixAdd (String A, String B) {
        int i = A.length() - 1;
        int j = B.length() - 1;
        StringBuilder sb = new StringBuilder(Math.max(A.length(), B.length()));
        boolean carry = false;

        while (i >= 0 && j >= 0) {
            int res = add(A.charAt(i), B.charAt(j));
            if (carry) res++;
            if (res >= 36) {
                carry = true;
                res -= 36;
            } else {
                carry = false;
            }
            sb.append(toChar(res));
            i--;
            j--;
        }

        int k = (i >= 0) ? i : j;
        String C = (i >= 0) ? A : B;
        while (k >= 0) {
            int res = add(C.charAt(k), '0');
            if (carry) res++;
            if (res >= 36) {
                carry = true;
                res -= 36;
            } else {
                carry = false;
            }
            sb.append(toChar(res));
            k--;
        }

        if (carry) sb.append(1);
        return sb.reverse().toString();
    }

    private int add(char a, char b) {
        int aa, bb;

        if (a >= '0' && a <= '9') {
            aa = a - '0';
        } else {
            aa = a - 'a' + 10;
        }

        if (b >= '0' && b <= '9') {
            bb = b - '0';
        } else {
            bb = b - 'a' + 10;
        }

        return aa + bb;
    }

    private char toChar(int res) {
        if (res <= 9) return (char) (res + '0');
        return (char) (res + 'a' - 10);
    }
}
```
