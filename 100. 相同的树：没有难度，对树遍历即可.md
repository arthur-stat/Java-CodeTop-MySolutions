题目简述：

> 给你两棵二叉树的根节点 `p` 和 `q` ，编写一个函数来检验这两棵树是否相同。
>
> 如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。

题目链接：[100. 相同的树](https://leetcode.cn/problems/same-tree/)

# 递归

没有难度，先序遍历即可。

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */

class Solution {
    public boolean isSameTree(TreeNode p, TreeNode q) {
        if (p == null && q == null) {
            return true;
        } else if (!(p != null && q != null)) {
            return false;
        }
        return dfs(p, q);
    }

    public boolean dfs(TreeNode p, TreeNode q) {
        if (p.val != q.val) return false;

        if (p.left != null) {
            if (q.left == null) return false;
            if (!dfs(p.left, q.left)) return false;
        } else if (q.left != null) {
            return false;
        }

        if (p.right != null) {
            if (q.right == null) return false;
            if (!dfs(p.right, q.right)) return false;
        } else if (q.right != null) {
            return false;
        }

        return true;
    }
}
```

# 迭代

改为迭代也是很容易的。

```java
class Solution {
    public boolean isSameTree(TreeNode p, TreeNode q) {
        if (p == null && q == null) {
            return true;
        } else if (!(p != null && q != null)) {
            return false;
        }

        Deque<TreeNode> sp = new ArrayDeque<>();
        Deque<TreeNode> sq = new ArrayDeque<>();
        sp.push(p);
        sq.push(q);

        while (!sp.isEmpty()) {
            TreeNode pp = sp.pop();
            TreeNode qq = sq.pop();

            if (pp.val != qq.val) return false;

            if (pp.right != null) {
                if (qq.right == null) return false;
                sp.push(pp.right);
                sq.push(qq.right);
            } else if (qq.right != null) {
                return false;
            }

            if (pp.left != null) {
                if (qq.left == null) return false;
                sp.push(pp.left);
                sq.push(qq.left);
            } else if (qq.left != null) {
                return false;
            }
        }

        return true;
    }
}
```
