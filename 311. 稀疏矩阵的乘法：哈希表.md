题目简述：

> 给定两个 [稀疏矩阵](https://en.wikipedia.org/wiki/Sparse_matrix) A 和 B，返回AB的结果。
> 您可以假设A的列数等于B的行数。

题目链接：[311. 稀疏矩阵的乘法](https://leetcode.cn/problems/sparse-matrix-multiplication/)

LintCode 题目链接：[654 · 稀疏矩阵乘法](https://www.lintcode.com/problem/654/)

# 思路

我想到的方案就是用哈希表记录 A 的非零行与 B 的非零列，然后在计算结果矩阵时查哈希表快速找到非零元素。

# 代码

```java
public class Solution {
    /**
     * @param a: a sparse matrix
     * @param b: a sparse matrix
     * @return: the result of A * B
     */
    public int[][] multiply(int[][] a, int[][] b) {
        int m = a.length;
        int n = a[0].length;
        int p = b[0].length;

        /* 用哈希表存储 a 的行与 b 的列的所有非零元素，方便后续计算 */
        // 非零元素的行为 key，列为 value
        Map<Integer, Set<Integer>> aRow = new HashMap<>();
        // 非零元素的列为 key，行为 value
        Map<Integer, Set<Integer>> bCol = new HashMap<>();

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (a[i][j] != 0) aRow.computeIfAbsent(i, k -> new HashSet<>()).add(j);
            }
        }

        for (int i = 0; i < n; i++) {
            for (int j = 0; j < p; j++) {
                if (b[i][j] != 0) bCol.computeIfAbsent(j, k -> new HashSet<>()).add(i);
            }
        }

        int[][] res = new int[m][p];
        for (int i = 0; i < m; i++) {
            Set<Integer> aRowSet = aRow.get(i);
            for (int j = 0; j < p; j++) {
                Set<Integer> bColSet = bCol.get(j);
                if (aRowSet != null && bColSet != null) {
                    /* LintCode 平台提供的 JDK 版本太低了，流式接口缺乏支持 */
                    // Set<Integer> c = aRowSet.stream().filter(bColSet::contains).toSet();
                    Set<Integer> c = new HashSet<>(aRowSet);
                    c.retainAll(bColSet);
                    for (int k : c) res[i][j] += a[i][k] * b[k][j];
                }
            }
        }

        return res;
    }
}
```
