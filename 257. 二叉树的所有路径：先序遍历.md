题目简述：

> 给你一个二叉树的根节点 `root` ，按 **任意顺序** ，返回所有从根节点到叶子节点的路径。
>
> **叶子节点** 是指没有子节点的节点。

题目链接：[257. 二叉树的所有路径](https://leetcode.cn/problems/binary-tree-paths/)

# 思路

本问题完全就是先序遍历的另一种描述。

因此直接先序遍历即可。

# 递归

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */

class Solution {

    private StringBuilder path;
    private List<String> res;

    public List<String> binaryTreePaths(TreeNode root) {
        // if (root == null) return List.of();
        path = new StringBuilder();
        res = new ArrayList<>();
        dfs(root);
        return res;
    }

    private void dfs(TreeNode node) {
        String s = node.val + "->";

        if (node.left == null && node.right == null) {
            res.add(path.toString() + node.val);
            return;
        }

        if (node.left != null) {
            path.append(s);
            dfs(node.left);
            path.setLength(path.length() - s.length());
        }

        if (node.right != null) {
            path.append(s);
            dfs(node.right);
            path.setLength(path.length() - s.length());
        }
    }
}
```

# 迭代

```java
class Solution {

    public List<String> binaryTreePaths(TreeNode root) {
        // if (root == null) return List.of();
        Deque<TreeNode> nodes = new ArrayDeque<>();
        Deque<String> paths = new ArrayDeque<>();
        List<String> res = new ArrayList<>();
        nodes.push(root);
        paths.push(String.valueOf(root.val));

        while (!nodes.isEmpty()) {
            TreeNode cur = nodes.pop();
            String path = paths.pop();

            if (cur.right == null && cur.left == null) {
                res.add(path);
                continue;
            }

            if (cur.right != null) {
                nodes.push(cur.right);
                paths.push(path + "->" + cur.right.val);
            }

            if (cur.left != null) {
                nodes.push(cur.left);
                paths.push(path + "->" + cur.left.val);
            }

        }

        return res;
    }
}
```
