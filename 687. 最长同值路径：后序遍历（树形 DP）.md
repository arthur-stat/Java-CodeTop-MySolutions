题目简述：

> 给定一个二叉树的 `root` ，返回 *最长的路径的长度* ，这个路径中的 *每个节点具有相同值* 。 这条路径可以经过也可以不经过根节点。
>
> **两个节点之间的路径长度** 由它们之间的边数表示。

题目链接：[687. 最长同值路径](https://leetcode.cn/problems/longest-univalue-path/)

# 思路

我记得此前我也曾解决过一道类似的问题，二者的共同点抽象后可以被概况为：递归的返回值不直接是答案，而是返回一个间接的值，最终的答案是在递归的过程中计算并更新的，而且答案的计算依赖于子递归调用的返回值。

我记得那个相似的问题也是这样，题目背景同样是二叉树的遍历，递归调用的返回值是无拐点的路径长度，递归体内答案的计算则同时考虑有无拐点的路径长度，而这依赖于子递归调用返回的子节点的无拐点路径长度。

---

所谓 “路径”，我们可以分为两类：

1. 存在拐点的路径，类似 $\Lambda$ 形；
2. 不存在拐点的路径，类似 $/$ 或 $\textbackslash$ 形。

我们定义：对于节点 $x$，记 $f(x)$ 表示以该节点最顶层节点的相同节点值路径的长度，我们称这样的节点为对应路径的 “起点”。

- 对于第一种路径的情形，起点为拐点；
- 对于第二种路径的情形，起点为上顶点。

最长同值路径要么是第一种路径，要么是第二种路径。因此，当我们遍历所有的节点并假设每个节点为起点，就能得到所有可能的起点对应的最长同值路径长度，最终可以得到全局最长同值路径长度。因此，在递归体内计算路径长度时，要同时考虑有无拐点的路径长度。

但在递归体内计算第一种路径（即有拐点的 $\Lambda$ 形）长度时，我们需要知道左右子节点分别作为起点的第二种路径（即无拐点路径）长度，这是因为：
$$
\text{第一种路径长度}=\text{同值左子节点第二种路径长度}+\text{同值右子节点第二种路径长度}+1
$$
也就是说，如果记第一种路径长度为 $g_1(x)$、第二种路径长度为 $g_2(x)$，则可能的全局最长同值路径为
$$
f(x)=\max\big\{g_1(x),g_2(x)\big\}
$$
而 $g_1(x)$ 的计算公式为
$$
g_1(x)=g_2(left)\times I_{\{x.val=left.val\}}+g_2(right)\times I_{\{x.val=right.val\}}+1
$$
$g_2(x)$ 的计算公式为
$$
g_2(x)=\max\big\{g_2(left)\times I_{\{x.val=left.val\}},\,g_2(right)\times I_{\{x.val=right.val\}}\big\}+1
$$
所以，递归体的返回值需要被设置为第二种路径的长度。

---

通过以上分析能看出来，我们需要先遍历左右子节点才能在父节点中计算路径长度，这天然是后序遍历的顺序。

故本问题考虑后序遍历，应在后序遍历的框架下设计算法。

---

这个其实是一种单状态树形 DP，只是数学建模太简单，所以我更倾向于将其归类为树的遍历。

# 代码

算法的时间复杂度为 $O(n)$，空间复杂度为 $O(h)$，其中空间开销主要来自于栈递归，$h$ 为树高。

注意 `res` 统计的是目标路径上节点的数量，而要求我们返回的是边的长度，因此要减 1。

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */

class Solution {

    private int res;

    public int longestUnivaluePath(TreeNode root) {
        res = 0;
        dfs(root);
        return Math.max(res - 1, 0);
    }

    private int dfs(TreeNode node) {
        if (node == null) return 0;

        int leftPathLen = dfs(node.left);
        int rightPathLen = dfs(node.right);
        int curvedPathLen = 1;
        int singlePathLen = 1;

        if (node.left != null && node.left.val == node.val) {
            singlePathLen = leftPathLen + 1;
            curvedPathLen += leftPathLen;
        }
        if (node.right != null && node.right.val == node.val) {
            singlePathLen = Math.max(singlePathLen, rightPathLen + 1);
            curvedPathLen += rightPathLen;
        }

        res = Math.max(res, curvedPathLen);

        return singlePathLen;
    }
}
```
