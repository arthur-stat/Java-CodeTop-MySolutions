题目简述：

> 给你一个正整数 `n` ，生成一个包含 `1` 到 `n2` 所有元素，且元素按顺时针顺序螺旋排列的 `n x n` 正方形矩阵 `matrix` 。

题目链接：[59. 螺旋矩阵 II](https://leetcode.cn/problems/spiral-matrix-ii/)

# 思路

可以按状态机模拟。

# 代码

```java
class Solution {
    public int[][] generateMatrix(int n) {
        int[][] res = new int[n][n];
        int count = 1;
        int circle = 0;  // 螺旋圈数
        int i = 0;  // row
        int j = 0;  // col
        int i_bound = -1;
        int j_bound = n - 1;
        char status = 'r';

        while (count < n * n) {
            res[i][j] = count;

            switch (status) {

                case 'r':
                    if (j < j_bound) {
                        j++;
                        count++;
                    } else {
                        status = 'd';
                        i_bound = n - circle - 1;
                    }
                    break;

                case 'd':
                    if (i < i_bound) {
                        i++;
                        count++;
                    } else {
                        status = 'l';
                        j_bound = circle;
                    }
                    break;
                
                case 'l':
                    if (j > j_bound) {
                        j--;
                        count++;
                    } else {
                        status = 'u';
                        i_bound = circle + 1;
                    }
                    break;

                case 'u':
                    if (i > i_bound) {
                        i--;
                        count++;
                    } else {
                        status = 'r';
                        circle++;
                        j_bound = n - circle - 1;
                    }
                    break;
            }
        }
        
        res[i][j] = n * n;

        return res;
    }
}
```
