题目简述：

> 给定两个等长字符串 `s` 和 `t` ，判断它们是否是同构的。
>
> 如果 `s` 中的字符可以按某种映射关系替换得到 `t` ，那么这两个字符串是同构的。
>
> 每个出现的字符都应当映射到另一个字符，同时不改变字符的顺序。不同字符不能映射到同一个字符上，相同字符只能映射到同一个字符上，字符可以映射到自己本身。

题目链接：[205. 同构字符串](https://leetcode.cn/problems/isomorphic-strings/)

# 模拟同构

## 思路

这是最暴力的解决方案，直接记录对两个字符串分别做字符索引做记录，其中字符索引就是字符到该字符所有索引集合的映射，然后判断索引集合们是否相等。即，针对两个字符串的字符索引统计哈希表，我们忽略其键，仅判断其整个值的集合是否相等，而每个哈希表的整个值的集合中都是若干个更小的字符索引集合。

这就是按题目描述的替换规则实现的模拟：忽略字符类型，对比字符的索引集。例如，假设在字符串 `s` 中映射为 $\big(a:(1,3),b:(2)\big)$，而在字符串 `t` 中映射为 $\big(a:(2),b:(1,3)\big)$，由于两个映射的值集合均为 $\big((1,3),(2)\big)$——此处忽略了键，所以我们就可以断言 `s` 与 `t` 互为所谓的同构字符串。

这个思路简单可行，但还不是最佳方案。

## 代码

算法的时间复杂度为 $O(n)$，空间复杂度为 $O(n)$。

```java
class Solution {
    public boolean isIsomorphic(String s, String t) {
        Map<Character, ArrayList<Integer>> ss = new HashMap<>();
        Map<Character, ArrayList<Integer>> tt = new HashMap<>();
        for (int i = 0; i < s.length(); i++) ss.computeIfAbsent(s.charAt(i), k -> new ArrayList<>()).add(i);
        for (int i = 0; i < t.length(); i++) tt.computeIfAbsent(t.charAt(i), k -> new ArrayList<>()).add(i);
        return new HashSet(ss.values()).equals(new HashSet(tt.values()));
    }
}
```

# 利用双射

## 思路

注意到按题目描述的映射实际上是一个一一映射（双射），所以可以分别维护两个映射：$s\to t$ 与 $t\to s$。

> - “每个出现的字符都应当映射到另一个字符”：满射
> - “相同字符只能映射到同一个字符上，不同字符不能映射到同一个字符上”：单射
>
> 二者结合，可以导出此处所谓的「同构」实际上只是一种双射关系。

我们可以尝试构造出这个双射。初始化两个空映射 $s\to t$ 与 $t\to s$，针对第 $i$ 个元素：

- 对于 `s`，$s\to t$ 应当能够使得 $s[i]\mapsto t[i]$，所以检查映射 $s\to t$ 是否能将 $s[i]$ 映射到 $t[i]$，若映射不存在则添加映射关系，若映射已存在且 $s[i]$ 被映射到其他元素则直接返回 `false`；
- 同理，对于 `t`，$t\to s$ 应当能够使得 $t[i]\mapsto s[i]$，所以检查映射 $t\to s$ 是否能将 $t[i]$ 映射到 $s[i]$，若映射不存在则添加映射关系，若映射已存在且 $t[i]$ 被映射到其他元素则直接返回 `false`。

之所以要同时满足上述两个映射关系，是为了保证映射是双射。

这是更优秀、更合适的解决方案，算法所需的空间开销与字符串长度无关。

## 代码

记字符集大小为 $|\Sigma|$，算法的时间复杂度为 $O(n)$，空间复杂度为 $O(|\Sigma|)$。

```java
class Solution {
    public boolean isIsomorphic(String s, String t) {
        if (s.length() != t.length()) return false;
        int n = s.length();

        Map<Character, Character> s2t = new HashMap<>();
        Map<Character, Character> t2s = new HashMap<>();

        for (int i = 0; i < n; i++) {
            char ss = s.charAt(i);
            char tt = t.charAt(i);

            Character c4s = s2t.get(ss);
            Character c4t = t2s.get(tt);

            if (c4s == null) s2t.put(ss, tt);
            else if (c4s != tt) return false;

            if (c4t == null) t2s.put(tt, ss);
            else if (c4t != ss) return false;
        }

        return true;
    }
}
```
