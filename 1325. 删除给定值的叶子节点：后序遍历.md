题目简述：

> 给你一棵以 `root` 为根的二叉树和一个整数 `target` ，请你删除所有值为 `target` 的 **叶子节点** 。
>
> 注意，一旦删除值为 `target` 的叶子节点，它的父节点就可能变成叶子节点；如果新叶子节点的值恰好也是 `target` ，那么这个节点也应该被删除。
>
> 也就是说，你需要重复此过程直到不能继续删除。

题目链接：[1325. 删除给定值的叶子节点](https://leetcode.cn/problems/delete-leaves-with-a-given-value/)

# 思路

既然删除子节点后父节点有可能变成叶子节点，从而父节点可能成为新的叶子节点而继续被删除，这意味着父节点必须要在所有的子节点被访问后才被访问，这不就是后序遍历的顺序嘛！所以本问题后序遍历即可解决。

# 代码

这是一个典型的递归实现的后序遍历算法，算法的时间复杂度为 $O(n)$，平均空间复杂度为 $O(\log n)$。

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */

class Solution {
    public TreeNode removeLeafNodes(TreeNode root, int target) {
        TreeNode dummy = new TreeNode(target - 1, root, null);
        dfs(dummy, target);
        return dummy.left;
    }

    public boolean dfs(TreeNode root, int target) {
        if (root == null) return false;

        if (dfs(root.left, target)) root.left = null;
        if (dfs(root.right, target)) root.right = null;

        return root.left == null && root.right == null && root.val == target;
    }
}
```
