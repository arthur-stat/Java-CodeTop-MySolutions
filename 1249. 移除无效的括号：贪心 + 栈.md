题目简述：

> 给你一个由 `'('`、`')'` 和小写字母组成的字符串 `s`。
>
> 你需要从字符串中删除最少数目的 `'('` 或者 `')'` （可以删除任意位置的括号)，使得剩下的「括号字符串」有效。
>
> 请返回任意一个合法字符串。
>
> 有效「括号字符串」应当符合以下 **任意一条** 要求：
>
> - 空字符串或只包含小写字母的字符串
> - 可以被写作 `AB`（`A` 连接 `B`）的字符串，其中 `A` 和 `B` 都是有效「括号字符串」
> - 可以被写作 `(A)` 的字符串，其中 `A` 是一个有效的「括号字符串」
>
> **提示：**
>
> - `1 <= s.length <= 10^5`
> - `s[i]` 可能是 `'('`、`')'` 或英文小写字母

题目链接：[1249. 移除无效的括号](https://leetcode.cn/problems/minimum-remove-to-make-valid-parentheses/)

# 思路

我们来总结一下初步思路：

- 算法的时间复杂度绝不可能达到甚至超过 $O(n^2)$，而本问题中也没有找到有序结构，所以我们要实现的算法时间复杂度应该是 $O(n)$；
- 英文小写字母实际上不影响结果，我们只需要关注括号字符的顺序。

> 根据这些信息，就可以排除掉动态规划了。那么可选项就剩下了线性扫描结构的算法。

这启发我们想到：进行一遍扫描，在扫描的过程中尽可能地、贪心地让括号们相匹配，最后再删掉没有能与之配对的括号们，剩下的字符所组成的字符串就是删除最少字符得到的括号字符串。这是因为，如果一对括号是能匹配上的，我们就可以忽略掉他们了；而无论我们用何种顺序匹配括号，最终需要删掉的失配括号数都是相同的，因此考虑最简单的贪心匹配。

不难推导，在贪心匹配的规则下，最终只有两种情况会导致括号失配：

1. 某个左括号 `'('` 的右侧不再有右括号 `')'`，该左括号将失配；
2. 某个右括号 `')'` 的左侧不再有左括号 `'('`，该右括号将失配。

除此之外的任何情况，一个括号都能与另一个括号相匹配。

---

具体操作时，使用一个左括号索引栈 `left` 与失配右括号索引数组 `right` 记录所有失配括号。我们进行一次扫描：

- 扫描到一个左括号 `'('` 时，将其索引压入栈 `left` 中，等到后续某个右括号 `')'` 与之匹配；
- 扫描到一个右括号 `')'` 时，先检查栈 `left` 是否为空，如果不为空则弹出栈顶，表示我们找到了一对匹配的括号；若 `left` 为空，则可以肯定该右括号失配，因为在贪心匹配的规则下没有能与之匹配的左括号，于是我们将该索引记录至 `right` 中。

这样，经过一次扫描后，`left` 中存储的就是失配左括号的索引、`right` 中存储的就是失配右括号的索引，于是删掉 `left` 与 `right` 中存储的索引对应的字符，所得到的新字符串就是删除最少字符后获得的括号字符串。

---

根据上述方案已经可以实现能 AC 的算法代码了，但以上方案里左括号与右括号的地位是不对等的。我们可以分别进行正向与反向两次扫描，每次扫描中只记录一种失配括号，即：

1. 正向扫描时，只记录失配右括号，因为失配左括号位于栈中，我们只有在扫描结束时才能肯定栈中存储的是失配括号的索引；
2. 反向扫描时，同理，只记录失配左括号。

这样，也能找出所有的失配括号。

# 代码

我们只按最初的思路实现算法，因为其他思路并没有从根本上优于该思路。

算法的时间复杂度为 $O(n)$，空间复杂度为 $O(n)$。

```java
class Solution {
    public String minRemoveToMakeValid(String s) {
        Deque<Integer> left = new ArrayDeque<>();  // stack
        Set<Integer> right = new HashSet<>();

        for (int i = 0; i < s.length(); i++) {
            char c = s.charAt(i);

            if (c == '(') {
                left.push(i);
            } else if (c == ')') {
                if (!left.isEmpty()) {
                    left.pop();
                } else {
                    right.add(i);
                }
            }
        }

        StringBuilder sb = new StringBuilder();
        Set<Integer> set = new HashSet<>(left);
        set.addAll(right);
        for (int i = 0; i < s.length(); i++) {
            if (!set.contains(i)) sb.append(s.charAt(i));
        }
        return sb.toString();
    }
}
```
