题目简述：

> 给定一个表示数据的整数数组 `data` ，返回它是否为有效的 **UTF-8** 编码。
>
> **UTF-8** 中的一个字符可能的长度为 **1 到 4 字节**，遵循以下的规则：
>
> 1. 对于 **1 字节** 的字符，字节的第一位设为 0 ，后面 7 位为这个符号的 unicode 码。
> 2. 对于 **n 字节** 的字符 (n > 1)，第一个字节的前 n 位都设为1，第 n+1 位设为 0 ，后面字节的前两位一律设为 10 。剩下的没有提及的二进制位，全部为这个符号的 unicode 码。
>
> 这是 UTF-8 编码的工作方式：
>
> ```
>       Number of Bytes  |        UTF-8 octet sequence
>                        |              (binary)
>    --------------------+---------------------------------------------
>             1          | 0xxxxxxx
>             2          | 110xxxxx 10xxxxxx
>             3          | 1110xxxx 10xxxxxx 10xxxxxx
>             4          | 11110xxx 10xxxxxx 10xxxxxx 10xxxxxx
> ```
>
> `x` 表示二进制形式的一位，可以是 `0` 或 `1`。
>
> **注意：**输入是整数数组。只有每个整数的 **最低 8 个有效位** 用来存储数据。这意味着每个整数只表示 1 字节的数据。

题目链接：[393. UTF-8 编码验证](https://leetcode.cn/problems/utf-8-validation/)

# 代码

就……按要求模拟好了。很标准的一个模拟类型的问题。

算法的时间复杂度为 $O(n)$，空间复杂度为 $O(1)$。

```java
class Solution {
    public boolean validUtf8(int[] data) {
        int len = 0;
        for (int code : data) {
            if (len == 0) {
                len = numberOfBytes(code);
                if (len == -1) return false;
            } else {
                if (!isValidSubcode(code)) return false;
            }
            len--;
        }
        return len == 0;
    }

    /**
     * 根据 UTF-8 编码的首个字节判断该字符的长度
     *
     * @param firstCode UTF-8 编码的首字节
     * @return 该 UTF-8 编码的字节长度，若字节序列无效则返回 -1
     */
    public int numberOfBytes(int firstCode) {
        if (((firstCode >> 7) & 1) == 0) return 1;
        if (((firstCode >> 6) & 1) == 0) return -1;

        for (int i = 5; i >= 3; i--) {
            if (((firstCode >> i) & 1) == 0) return 7 - i;
        }

        return -1;
    }

    public boolean isValidSubcode(int code) {
        return (((code >> 7) & 1) == 1) && (((code >> 6) & 1) == 0);
    }
}
```
