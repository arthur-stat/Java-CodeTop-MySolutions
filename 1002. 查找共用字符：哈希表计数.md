题目简述：

> 给你一个字符串数组 `words` ，请你找出所有在 `words` 的每个字符串中都出现的共用字符（**包括重复字符**），并以数组形式返回。你可以按 **任意顺序** 返回答案。

题目链接：[1002. 查找共用字符](https://leetcode.cn/problems/find-common-characters/)

# 思路

扫描所有单词，对每个单词进行词频统计。考虑两个词频计数哈希表，一个用于存储全局每个字符在各单词中的最小公共词频，一个用于存储统计当前单词中每个字符的词频。每当统计完一个单词，更新全局词频哈希表，更新方式为取全局词频哈希表和对应字符的当前单词词频哈希表的最小值。这样，最终就能得到每个字符在各个单词中的最小公共词频。

# 代码

算法的时间复杂度为 $O(n)$，空间复杂度为 $O(|\Sigma|)$，其中 $n$ 是所有单词的所有字符的数量，$|\Sigma|$ 是字符集的大小——在本问题中，$|\Sigma|$ 是常数 26。

```java
 class Solution {
    public List<String> commonChars(String[] words) {
        int[] freq = new int[26];
        Arrays.fill(freq, Integer.MAX_VALUE);
        int[] freq4word = new int[26];
        
        for (String word : words) {
            for (char c : word.toCharArray()) freq4word[c - 'a']++;
            for (int i = 0; i < 26; i++) freq[i] = Math.min(freq[i], freq4word[i]);
            Arrays.fill(freq4word, 0);
        }

        List<String> res = new ArrayList<>();
        for (int i = 0; i < 26; i++) {
            for (int j = 0; j < freq[i]; j++) {
                res.add(Character.toString((char) (i + 'a')));
            }
        }
        return res;
    }
}
```
