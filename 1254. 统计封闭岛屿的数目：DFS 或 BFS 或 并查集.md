题目简述：

> 二维矩阵 `grid` 由 `0` （土地）和 `1` （水）组成。岛是由最大的4个方向连通的 `0` 组成的群，封闭岛是一个 `完全` 由1包围（左、上、右、下）的岛。
>
> 请返回 *封闭岛屿* 的数目。

题目链接：[1254. 统计封闭岛屿的数目](https://leetcode.cn/problems/number-of-closed-islands/)

# 思路

这道题和 [200. 岛屿数量](https://leetcode.cn/problems/number-of-islands/) 几乎没什么区别吧？主要的差异在于这里当且仅当陆地被海水完全包围时才算为一个封闭岛屿，修改计数的条件即可。

考虑 DFS、BFS 或 并查集。

# DFS

算法的时间复杂度为 $O(mn)$，空间复杂度为递归深度，最坏为 $O(mn)$。这里通过原地更新 `grid` 实现染色标记，从而避免了使用额外的 `visited` 数组记录访问状态。也可以很轻易地改为显式栈的实现。

```java
class Solution {
    public int closedIsland(int[][] grid) {
        int m = grid.length;
        int n = grid[0].length;
        int res = 0;

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == 0) {
                    if (dfs(grid, i ,j)) res++;
                }
            }
        }

        return res;
    }

    // condition: grid[i][j] = 0
    private boolean dfs(int[][] grid, int i, int j) {
        if (i < 0 || i >= grid.length || j < 0 || j >= grid[0].length) return false;
        if (grid[i][j] == 1) return true;

        grid[i][j] = 1;
        boolean r1 = dfs(grid, i - 1, j);
        boolean r2 = dfs(grid, i + 1, j);
        boolean r3 = dfs(grid, i, j - 1);
        boolean r4 = dfs(grid, i, j + 1);
        return r1 && r2 && r3 && r4;
    }
}
```

# BFS

算法的时间复杂度为 $O(mn)$，空间复杂度为 $O(mn)$，其中空间复杂度主要来自队列。

```java
class Solution {
    public int closedIsland(int[][] grid) {
        int m = grid.length;
        int n = grid[0].length;
        int res = 0;

        Deque<Long> queue = new ArrayDeque<>();
        boolean[][] visited = new boolean[m][n];

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == 0 && !visited[i][j]) {
                    queue.offer(pkg(i, j));
                    boolean flag = true;
                    while (!queue.isEmpty()) {
                        long z = queue.poll();
                        int ii = (int) (z >>> 32);
                        int jj = (int) z;
                        if (ii < 0 || ii >= m || jj < 0 || jj >= n) {
                            flag = false;
                            continue;
                        }
                        if (grid[ii][jj] == 1 || visited[ii][jj]) continue;
                        visited[ii][jj] = true;
                        queue.offer(pkg(ii - 1, jj));
                        queue.offer(pkg(ii + 1, jj));
                        queue.offer(pkg(ii, jj - 1));
                        queue.offer(pkg(ii, jj + 1));
                    }
                    if (flag) res++;
                }
            }
        }

        return res;
    }

    public long pkg(int a, int b) {
        return (((long) a) << 32) | (b & 0xFFFFFFFFL);
    }
}
```

# 并查集

一个封闭岛屿内的点视为隶属于同一个分类集合。

算法的时间复杂度为 $O(mn)$，空间复杂度为 $O(mn)$。

```java
class UnionFind {

    private int[] parent;
    private int[] rank;
    private boolean[] isEdge;
    private int count;

    public UnionFind(int n, int rows, int cols) {
        parent = new int[n];
        rank = new int[n];
        isEdge = new boolean[n];
        count = 0;
        
        for (int i = 0; i < n; i++) {
            parent[i] = i;
            int r = i / cols;
            int c = i % cols;
            if (r == 0 || r == rows - 1 || c == 0 || c == cols - 1) {
                isEdge[i] = true;
            }
        }
    }

    public int find(int x) {
        if (parent[x] != x) {
            parent[x] = find(parent[x]);
        }
        return parent[x];
    }

    public void union(int x, int y) {
        int rootX = find(x);
        int rootY = find(y);
        
        if (rootX == rootY) return;
        
        if (rank[rootX] < rank[rootY]) {
            parent[rootX] = rootY;
            isEdge[rootY] = isEdge[rootX] || isEdge[rootY];
        } else if (rank[rootX] > rank[rootY]) {
            parent[rootY] = rootX;
            isEdge[rootX] = isEdge[rootX] || isEdge[rootY];
        } else {
            parent[rootY] = rootX;
            isEdge[rootX] = isEdge[rootX] || isEdge[rootY];
            rank[rootX]++;
        }
    }
    
    public void addLand(int index) {
        count++;
    }
    
    public boolean isEdge(int index) {
        return isEdge[find(index)];
    }
    
    public int getCount() {
        return count;
    }
}

class Solution {
    public int closedIsland(int[][] grid) {
        if (grid == null || grid.length == 0) return 0;
        
        int m = grid.length;
        int n = grid[0].length;
        UnionFind uf = new UnionFind(m * n, m, n);
        
        int[][] directions = {{0, 1}, {1, 0}, {0, -1}, {-1, 0}};
        
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == 0) {
                    int idx = i * n + j;
                    
                    for (int[] dir : directions) {
                        int ni = i + dir[0];
                        int nj = j + dir[1];
                        
                        if (ni >= 0 && ni < m && nj >= 0 && nj < n && grid[ni][nj] == 0) {
                            int nIdx = ni * n + nj;
                            uf.union(idx, nIdx);
                        }
                    }
                }
            }
        }
        
        boolean[] counted = new boolean[m * n];
        int closedCount = 0;
        
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == 0) {
                    int idx = i * n + j;
                    int root = uf.find(idx);
                    
                    if (!counted[root] && !uf.isEdge(root)) {
                        closedCount++;
                        counted[root] = true;
                    }
                }
            }
        }
        
        return closedCount;
    }
}
```
